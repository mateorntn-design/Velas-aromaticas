# Velas Aromáticas

## Project Description
This project focuses on creating aromatic candles using natural ingredients, promoting sustainability and wellness. Our aim is to provide a range of soothing scents that enhance the home environment and foster relaxation.

## GitHub Pages Setup Instructions
1. Go to the repository settings.
2. Scroll down to the "GitHub Pages" section.
3. Choose the source branch as `main` and click on "Save".
4. Your GitHub Pages site will be published at `https://<your-username>.github.io/<repo-name>`.

### Note
Make sure to select `main` as the branch for the GitHub Pages source.
